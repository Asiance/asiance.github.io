Time: about 3h.

Part I: Create a REST API

Design a REST API that provides author's posts

Requirements

An author is defined as follows:

  • name
  • role
  • avatar
  • location
  • created_at
  • updated_at

A post is defined as follow:

  • author
  • title
  • body
  • tags (can be empty)
  • image_url
  • created_at
  • updated_at

Part II: Create a webpage
