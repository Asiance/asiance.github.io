ASIANCE coding challenge - Front/backend developer

Go back

Time: about 60 minutes recommended.

Recreate the following reference image in HTML and CSS using a templating library.

If you are applying as a a Fullstack Engineer do both the frontend and the backend.

If you are applying as a a Frontend developer do only the frontend.

If you are applying as a a Backend developer do only the backend.

Requirements

An author is defined as follow:

  • name
  • role
  • avatar
  • location

A post is defined as follow:

  • author
  • title
  • body
  • tags (can be empty)
  • image url
If you apply for the frontend position only please use these 2 JSON files, otherwise pull the data from your backend source: authors and posts.

You can use a framework.

Optional

  • About us page
  • Functional pagination

Programming Language

Feel free to choose the one you prefer (Asiance is mainly using PHP, JavaScript and Python).
